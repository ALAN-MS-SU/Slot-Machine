# Slot Machine 🎰

A slot machine project focused on smooth animations, custom spin logic, and a modern UI built with current web technologies.

---

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Features

- Animated spinning system
- Randomized custom values
- Responsive interface
- Reusable components
- State and repeat control
- Modern styling with Tailwind CSS

---

## Project Structure

```bash
src/
 ├── Components/
 ├── Lib/
 ├── Types/
 ├── App/
 └── Assets/
```

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/ALAN-MS-SU/Slot-Machine.git
```

Navigate to the project folder:

```bash
cd Slot-Machine
```

Install dependencies:

```bash
yarn install
```

Run the development server:

```bash
yarn dev
```
